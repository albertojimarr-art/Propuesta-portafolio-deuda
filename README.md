
# AJ | Generador visual de propuesta de inversión

Esta versión genera la página 1 como una imagen de alta resolución y la exporta a PDF.  
El resultado visual es más cercano a las imágenes tipo infografía generadas en el chat.

## Archivos
- `app.py`
- `requirements.txt`

## Deploy en Streamlit Cloud
1. Sube estos archivos a GitHub.
2. En Streamlit Cloud selecciona tu repositorio.
3. Main file path: `app.py`.
4. Deploy.

## Nota
El PDF se genera a partir de una imagen para mantener diseño fijo.  
Esto mejora la apariencia visual, pero el texto dentro del PDF no será editable como texto vectorial.
